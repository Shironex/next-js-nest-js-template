## Welcome

This is a comprehensive monorepo template featuring a NestJS API backend and Next.js frontend with shadcn/ui components. The project is structured as a Turborepo workspace using pnpm for dependency management.

## Features

### Backend (NestJS API)
- **Authentication**: Complete session-based auth with email verification
- **Security**: Rate limiting, CSRF protection, and account lockout
- **File Uploads**: AWS S3 integration with local development support
- **Subscriptions**: Full Stripe integration for payment processing
- **Email**: React Email templates with Mailpit for development
- **Database**: PostgreSQL with Prisma ORM
- **Testing**: Comprehensive unit and e2e test coverage

### Frontend (Next.js)
- **Modern Stack**: Next.js 15 with React 19
- **UI Components**: shadcn/ui with Tailwind CSS
- **State Management**: React Query for server state
- **Forms**: React Hook Form with Zod validation
- **Authentication**: Complete auth flows with session management
- **Stripe**: Subscription management and billing portal

### Development Experience
- **Monorepo**: Turborepo for fast, efficient builds
- **Type Safety**: Full TypeScript coverage
- **Code Quality**: ESLint, Prettier, and pre-commit hooks
- **Docker**: Complete containerization for all services
- **Documentation**: This comprehensive documentation site!
